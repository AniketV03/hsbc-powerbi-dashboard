# 📊 HSBC Bank Customers Dashboard (2015)

## 🔍 Overview
This Power BI project analyzes HSBC bank customer data for the year 2015. It provides insights into customer demographics, balances, job classifications, and regional distribution.

## 📊 Key Insights
- England has the highest number of customers and total balance
- White collar customers hold the highest financial value
- Majority customers are in 26–45 age group
- Customer distribution increases steadily across quarters
- Female and male contribution is balanced across segments
